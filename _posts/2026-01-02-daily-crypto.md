MiCA covers stablecoins (ART/EMT), spot trading, custody, exchange services; harmonizes AML/market-abuse rules EU-wide.[^60][^43][^47][^46]

## Security Incidents

1. **Trust Wallet browser extension exploit (Dec 24, 2025; \$7M stolen)**[^34][^33][^35]
    - Malicious code injected into Trust Wallet Chrome extension v2.68 during Christmas update; ~\$7M drained from user wallets. Trust Wallet CEO Eowyn Chen confirmed full reimbursement and removal from Chrome Web Store; improved version released with ownership-verification tools.[^33][^35]
    - **On-chain confirmation**: PeckShield tracked exploit; Trust Wallet official statement (Dec 26–Jan 1).[^35][^33]
2. **Address-poisoning scam (December 2025; \$50M loss)**[^34][^33][^35]
    - Single victim sent funds to lookalike wallet address (address poisoning) in largest December exploit. PeckShield: scammers use similar-looking addresses to trick users into sending payments to wrong destination.[^33][^35]
    - **Mitigation**: Always verify full address character-by-character; use address book features; test small amounts first.
3. **Multi-signature wallet private-key leak (December 2025; \$27.3M loss)**[^34][^33]
    - Private key associated with multi-sig wallet compromised, resulting in \$27.3M theft. PeckShield warns key-management vulnerabilities persist even for wallets requiring multiple authorizations.[^33][^34]
4. **Binance BROCCOLI(714) market manipulation (Jan 1, 2026; bug or hack)**[^58]
    - Market-maker account on Binance allegedly hacked or exploited via bug; attacker aggressively bought low-liquidity BROCCOLI(714) spot while opening leveraged perpetual positions to pump price and transfer value. Trader "Vida" detected anomaly with automated alerts, profited ~\$1M by front-running pump then shorting. Binance has not issued official statement; highlights exchange risk-control gaps.[^58]
5. **Mystery EVM-chain wallet-draining attack (Jan 1, 2026; \$107K+ stolen)**[^33]
    - Unexplained exploit hit Ethereum and EVM-compatible chains; possible link to Trust Wallet hack (code injection vectors similar). Investigation ongoing; no clear source identified as of Jan 2 08:00 UTC.[^33]

**Total December 2025 exploits**: 26 incidents, \$76M losses (–60% vs. November's \$194.3M). Full-year 2025 theft: \$3.4B+ (Jan–Nov data).[^37][^36][^35][^34][^33]
